Event Specific Guidelines

1. Online Registration is Mandatory: All participants must register online to take part in the contest.

2. Problem Set: Participants will receive a set of coding problems to solve within the time limit.

3. Programming Languages: The competition will support a predefined set of programming languages: C, C++, Java, and Python.

4. Number of Questions: Each participant will tackle a total of 5 questions: 1 hard, 3 medium, and 1 simple.

5. Time Duration: The competition will typically last for 1 hour.

6. Code Correctness: Participants must write code that compiles and runs correctly. Points will be deducted for syntax errors, runtime errors, or any code that fails to meet the problem requirements.

7. Submission Guidelines: Participants must submit their completed code before the deadline; late submissions will not be accepted.

8. No Code Sharing: Sharing or copying code from other participants is strictly prohibited and will lead to disqualification.

9. Integrity Policy: Any attempt to manipulate the competition system or cheat will result in immediate disqualification.

10. Judges’ Decisions: The decisions made by the judges are final, and any disputes must be addressed immediately after the competition concludes.
